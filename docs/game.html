    • randomSeed

      Declaration

      immutable long randomSeed;

      Return Value

      Возвращает некоторое число, которое ваша стратегия может использовать для инициализации генератора случайных чисел. Данное значение имеет рекомендательный характер, однако позволит более точно воспроизводить прошедшие игры.

    • tickCount

      Declaration

      immutable int tickCount;

      Return Value

      Возвращает базовую длительность игры в тиках. Реальная длительность может отличаться от этого значения в меньшую сторону. Эквивалентно world.tickCount.
